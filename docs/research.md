All three platforms (YouTube, Instagram, Meta) provide **substantial FREE analytics** in 2026. Here's what they offer and where the gaps are.

## Instagram Insights

### ‚úÖ What Creators Get (FREE)

**Content Performance:**

- Likes, comments, **"Sends per Reach"** (shares), saves
- **Views** (universal metric across posts/stories/reels)
- Precise engagement timing (when a reel was liked, which carousel slide engaged)

**Audience:**
- Followers' age, gender, top locations
- Most active times

**Reels Focus:**
- Reels = **70% of Instagram views** in 2026
- Dedicated Reels analytics

### ‚ö†Ô∏è Limitations (2026 Changes)

**Data Retention: Only 90 days!**
- Creators MUST export data regularly or lose history
- No long-term trend analysis without manual exports

**No PDF Reports:**
- Instagram is "developing" PDF summaries for brand partnerships
- Not yet available

### üîì API Access

**Instagram Graph API** allows:
- Access to their own insights (impressions, reach)
- Manage posts, stories, comments
- **NO** access to competitors or other users

**Restrictions:**
- Basic Display API retired (Dec 2024)
- No public hashtag/discovery data
- 90-day data retention limit also applies to API
